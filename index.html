What is FlameCord?

FlameCord is an antibot BungeeCord and Waterfall fork with antibot features, exploit prevention systems, performance improvements, removal of unused features, library updates and application layer (L7) ddos mitigation.

How do I install it?

FlameCord is easy to install and use. Just download the file from BuiltByBit and drag an drop FlameCord.jar into your server folder and run it.
